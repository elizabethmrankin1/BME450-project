# Title

Exercise Form Correction AI Helper

## Team members

Elizabeth Rankin (elizabethmrankin1), Kylie Graf (Kyliegraf)

## Project decription

The goal our project is to create a AI based form correction trainer that will tell an athlete if their form for basic excercises is correct or incorrect based on an image input while they are doing the excersise. To begin, we plan on training the model to identify proper squat form. We plan to identify proper squat form as this is one of the easiest to identify. After creating a model that successfully inform the user if their form is correct or not, we can perform iterations and make it more complex. Keeping the squat as the execsize, if the user has inccorect form we can tell them how it is incorect. Another improvement we could make is add aditional excersises such as a plank or push up. The dataset we plan to use is https://zenodo.org/records/17558630. We plan to use this dataset because it consists of three classes of form: Correct, Incorrect Posture, and Incorrect Foot Placement. It is a sufficiently large dataset (hundreds of images for each category) and is pre-labeled.
