# Title

Excersie Form Correction AI Helper

## Team members

Elizabeth Rankin (elizabethmrankin1), NameB (kyliegraf)

## Project decription

The goal our project is to create a AI based form correction trainer that will tell an athlete if their form for basic excercises is correct or incorrect based on an image input while they are doing the excersise. To begin, we plan on training the model to identify proper squat form. The dataset we plan to use is https://zenodo.org/records/17558630. We plan to use this dataset because it consists of three classes of form: Correct, Incorrect Posture, and Incorrect Foot Placement. It is a sufficiently large dataset (hundreds of images for each category) and is pre-labeled.
